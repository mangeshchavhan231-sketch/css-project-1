# YouTube Clone UI

A responsive YouTube homepage clone built using HTML and CSS.

## Features

- Responsive video grid layout
- Fixed header navigation
- Sidebar menu
- Search bar with tooltips
- Notification badge
- Responsive design using CSS Grid and Flexbox
- Hover effects and styling similar to YouTube

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Roboto)
